Java有8种基本数据类型: `byte`、`short`、`int`、`long`、`char`、`float`、`double`、`boolean`，分类如下:

* **整数类型**: byte、short、int、long、char
* **浮点类型**: float、double
* **布尔类型**: boolean
* **数值类型**: 包括整数类型和浮点类型

| 类型 | 位数 | 包装类 |
| ---       | ---   | ---   |
| byte      | 8位   | Byte    |
| short     | 18位  | Short  |
| int       | 32位  | Integer  |
| long      | 64位  | Long  |
| char      | 16位  | Character  |
| float     | 32位  | Float  |
| double    | 64位  | Double  |
| boolean   | 32位  | Boolean  |
