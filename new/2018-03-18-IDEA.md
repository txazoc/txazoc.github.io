#### 快捷键

* Recent Files: `Command + E`
* Class Search: `Command + N`
* File Search: `Shift + Command + N`
* Symbol Search: `Option + Shift + Command + N`
* Find: `Command + F`
* Replace: `Command + R`
* Find in Path: `Shift + Command + F`
* Find in Path: `Shift + Command + R`

#### 快捷操作

* psvm: main方法

```java
public static void main(String[] args) {
}
```

* n.fori: for循环

```java
for (int i = 0; i < n; i++) {
}
```

* sout: 标准输出

```java
System.out.println();
```
